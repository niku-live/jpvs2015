##01 tema - Konsolinës aplikacijos

Konsolinë aplikacija, tai áprasta kompiuteriams skirta programa, kurios naudotojo sàsaja paremta dialogu principu valdomu ávedant tekstà klaviatûra.

#Trumpai
Konsoliniø aplikacijoms sukurti Visual Studio 2013 aplinkoje pasirinkite File -> New -> Project... ir ið siûlomø projektø ðablonø pasirinkite Visual C# -> Windows Desktop -> Console Application
