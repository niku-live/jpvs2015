##03 tema - WPF aplikacijos

WPF (Windows Presentation Foundation) aplikacijos - tai Windows operacinei sistemai skirtos programos paremtos langais ir daug kuo panaðios á Windows Forms aplikacijas, taèiau viskà praplëèia papildomomis funkcijomis (animacija ir pan.)

#Trumpai
WPF aplikacijoms sukurti Visual Studio 2013 aplinkoje pasirinkite File -> New -> Project... ir ið siûlomø projektø ðablonø pasirinkite Visual C# -> Windows Desktop -> WPF Application
